# README

## Konfiguráció

Hozz létre egy `.env` nevű fájlt a `backend` mappában. Egy példa a fájl tartalmára:

``` txt
DB_PORT=1433
DB_USER=sa
DB_PASSWORD=Faszom1326
DB_NAME=tuning
DB_HOST=localhost
DB_INSTANCE=SQLEXPRESS
BACKEND_PORT=3000
```

## Futtatás

Navigálj el a `backend` mappába és futtasd az `npm start` parancsot. Ez úgy fogja elindítani a programot, hogy minden mentésnél autómatikusan újraindul.
